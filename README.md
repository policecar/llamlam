# ReadMe

Experiments with small language models.


## Installation

```bash
# Make virtual env
uv venv
source .venv/bin/activate

# Install Python packages
uv pip install -U pip
uv pip install -r requirements.txt

# Install pre-commit hooks
pre-commit install
```

## Usage

```
python -m llamlam.train
```
